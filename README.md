# MyBrain - Student Flow AI

> 智能学生任务流管理系统

## 核心理念

MyBrain 是一款专为学生设计的智能任务管理应用，采用"三轨分类法"实现任务的精准调度与动态重排。

## 视觉规范

严格对标 **Things 3** 设计语言：

- **大留白**：界面呼吸感，信息密度适中
- **0.5px 极细线条**：精致而不喧宾夺主
- **非线性动画**：`cubic-bezier(0.25, 0.1, 0.25, 1)` 质感过渡
- **色彩体系**：柔和中性色为主，关键操作点缀品牌色

## 存储架构

**本地 SQLite 优先**，隐私数据不离端：

- 所有用户数据存储在本地 SQLite 数据库
- 无需网络连接即可完整使用
- 数据完全由用户掌控

## 业务逻辑：三轨分类法

### 轨道定义

| 轨道 | 代号 | 特征 | 示例 |
|------|------|------|------|
| 冲刺轨 | SPRINT | 短期高强度，明确截止日 | 明天的作业、本周考试 |
| 马拉松轨 | MARATHON | 长期持续，无硬性截止 | 背单词、健身计划 |
| 悬念轨 | SUSPENSE | 等待外部条件，暂不可执行 | 等老师回复、等材料到货 |

### 核心能力

- **实时动态重排**：根据完成情况、时间压力自动调整任务顺序
- **自动纠错自愈**：检测异常状态并自动修复（如过期未完成任务）
- **智能提醒**：基于任务轨道和紧急度的差异化提醒策略

## 平台适配

深度适配 **vivo 手机**：

- 后台常驻运行优化
- 电池管理白名单引导
- 通知权限精准申请
- OriginOS 系统特性适配
